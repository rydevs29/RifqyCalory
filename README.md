# RifqyKalori

RifqyKalori adalah aplikasi web PWA (Progressive Web App) untuk scan makanan dan minuman Indonesia menggunakan OCR (Tesseract.js), menghitung kalori, karbohidrat, protein, dan lemak secara otomatis. 100% offline, gratis, dan siap install di HP!

## Fitur Utama
- **Scan Foto Makanan/Minuman** â€“ Gunakan kamera HP untuk foto menu atau makanan, AI baca nama otomatis.
- **Database 280+ Item** â€“ 200+ makanan (nasi goreng, rendang, sate, dll) + 80+ minuman (es teh, kopi susu, jus jeruk, dll).
- **Hitung Gizi Instan** â€“ Kalori, karbo, protein, lemak + grafik pie chart.
- **Riwayat Makan** â€“ Simpan scan harian, hapus per item atau all.
- **Offline 100%** â€“ Kerja tanpa internet setelah pertama kali load.
- **PWA** â€“ Install di HP seperti app native, no install dari Play Store.
